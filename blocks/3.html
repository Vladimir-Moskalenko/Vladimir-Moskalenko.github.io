#3: New Siri, Old Siri

macOS Sequoia is a great update, but the new siri icon and UI aren't available on intel... tryna fix that!

So I stumbled upon a way to enable the new Siri UI, using the command "defaults write com.apple.assistant bypassDeviceSupportsSAE -bool true". THe command tells macOS to enable the new Apple Assistant features on any Mac.

Does it work? Partially. Not all Apple Intelligence files are in place, so the range of changes will vary. On Apple Silicon Macs, the new UI files are there, so it changes the UI and the icon. On intel... something goes wrong.

All it does on intel is... hide the siri animation. No other changes.

How to get the new icon on intel Macs (useless and will NOT work on Apple Silicon macs!)

1. Gain read/write access to your system by following this tutorial

2. Download the following file: Assets.car. I manually changed the siri icon so you won't have to. Or if you want a linear-styled icon, download thsi file: linear.car (don't forget to rename it to Assets.car before replacing the system icon)

3. Open Finder and place the file you got above into the folder: "~/lvmnt/System/Library/CoreServices/Siri.bundle/Contents/Resources/". You can press ⇧+⌘+G in Finder to quickly go there

4. Apply the changes: in Terminal, type the command: "sudo bless --mount ~/lvmnt --bootefi --create-snapshot"

5. Now reboot! You should see the new siri icon in your menubar (not UI, will look into that later)
