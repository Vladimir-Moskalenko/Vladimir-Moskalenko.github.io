#3: New Siri, Old Siri

macOS Sequoia is a great update, but the new siri icon and UI aren't available on intel... tryna fix that!

So I stumbled upon a way to enable the new Siri UI, using the command "defaults write com.apple.assistant bypassDeviceSupportsSAE -bool true". THe command tells macOS to enable the new Apple Assistant features on any Mac.

Does it work? Partially. Not all Apple Intelligence files are in place, so the range of changes will vary. On Apple Silicon Macs, the new UI files are there, so it changes the UI and the icon. On intel... something goes wrong.

All it does on intel is... hide the siri animation. No other changes.

How to get the new icon on intel Macs (useless and will NOT work on Apple Silicon macs!)

1. Disable SIP: reboot into Recovery, open the Terminal (Utilites > Terminal in the menubar), enter the following commands "csrutil disable && csrutil authenticated-root disable". Then reboot.

2. Get your disk ID: in macOS, open the Terminal and enter the following command: "diskutil list". In the list, choose your macOS partition name, and remember the diskXsX. Further on, replace diskXsX with the one you saw here.

3. Get the livemount: in Terminal, type the following command: "mkdir ~/lvmnt && sudo mount -o nobrowse -t apfs /dev/diskXsX ~/lvmnt"

4. Download the following file: Assets.car. I manually changed the siri icon so you won't have to.

5. Open Finder and place the file you got above into the folder: "~/lvmnt/System/Library/CoreServices/Siri.bundle/Contents/Resources/". You can press ⇧+⌘+G in Finder to quickly go there

6. Apply the changes: in Terminal, type the command: "sudo bless --mount ~/lvmnt --bootefi --create-snapshot"

7. Now reboot!
