#8: Project LicK

So here we are, school and college just started, my other classes didn't though, which does free me some time. Lately, I've been busy just chillaxing. But now it's time for some fun :)

Introducing... Project LicK - an attempt to make modern macOS look nice.

LicK?

"We made the buttons on the screen look so good you'll want to lick them." - Steve Jobs

And truly, he did. Moreover, for more than the next decade, Apple's UIs were just like that. Apparently, Tim Cook didn't entirely agree with Steve Jobs, and flattende down the UI in iOS 7, and later in OS X Yosemite.

People didn't like that UI, or, rather, they got bored of it. One of these people was me, and on one day, I discovered AquaLickX - a project by The One and Only MacOS Customizer Allan Nyholm!

This project originally was written for High Sierra (?), then ported to Catalina, Big Sur, Monterey. The latest supported version was Sonoma, that's what I'm going to use for the base OS

Why a project? Because I'm planning on doing way more, including customizing as much app icons as possible, and even writing an auto-install shell script. And since I may want to switch back to the default OS style, I'm going to make a separate partition and, if everything goes well, may even share a bootable iso!

The Plan (and current progress)

☑ Backups for safety

◻ Create a LicK partition (not an APFS volume, because I want to keep my current install safe and sound)

◻ Install macOS Sonoma (+ bootable installer)

◻ AquaLickX (14.5 edition)

◻ Custom app icons!

◻ More editing of system icons

◻ .sh file – so it's easier to do stuff after updates, plus so you could do the same!
