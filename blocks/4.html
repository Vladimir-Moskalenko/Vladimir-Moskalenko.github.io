The first macOS version since Monterey, Sequoia has an accessible wallpaper/screensaver file! Let's edit it together!

First, gain read/write access to your system by following this tutorial

The wallpaper isn't stored, like normally in "/System/Library/Desktop Pictures", it's stored in "/System/Library/ExtensionKit/Extensions/WallpaperHeliosExtension.appex". Be sure to go to that path in the ~/lvmnt folder!

Now, right-click the WallpaperHeliosExtension.appex file and select "Show Package Contents". Go to Contents/Resources. You're in!

What do all these files mean? The .exr files are the ones you will want to edit. They hold the data for the gradients used in the wallpapers; the .usdc files store the data of the camera path and the trees, while the rest of the files store the data of how everything works together.

You can try to manually edit the .exr files yourself, using a third-party painting tool or just by editing them in Preview.
