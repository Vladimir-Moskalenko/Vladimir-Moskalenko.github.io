For the first time: What macOS lacks

This time, unlike previous times when I was fighting for my life, I actually was pretty chill with Ubuntu. And not that chill with macOS now that I started using it again.

Here's a list of what macOS would have to become the best OS

1. Updates without reboot. Yes, macOS updates are good, but still not as great as Linux's, where you can update without rebooting, and the reboot (to apply them) takes like 5 seconds

2. Built-in package managers. I really got used to apt, and feel that is what macOS lacks. Homebrew is an alternative, but, once again, it requires manual setup

3. Acceleration control. Pointer acceleration is what is very frustrating at times. Being able to control the acceleration would make macOS feel better

4. One-click Launchpad. On Ubuntu, press the super (I remapped it to left-ctrl) key twice to view all apps. I'm on a touchbar Mac, so there isn't any Launchpad button by default, and extra trackpad gestures take up more time.
